RootWork Framework Design System

RW Container

The .rw-container class provides a max-width of 1200px with responsive margins:

  • Mobile: 1.5rem vertical, 0.75rem horizontal padding
  • Desktop (768px+): 1.5rem vertical, 2rem horizontal padding

Usage Examples

HTML Structure

<!-- Container -->
<div class="rw-container">
  <h1>Page Content</h1>
</div>

<!-- Grid Layout -->
<div class="rw-container">
  <div class="rw-grid">
    <div class="rw-col-4">Column 1</div>
    <div class="rw-col-4">Column 2</div>
    <div class="rw-col-4">Column 3</div>
  </div>
</div>

<!-- Using Spacing Tokens -->
<div style="padding: var(--space-6); margin-bottom: var(--space-8);">
  Content with consistent spacing
</div>

Available Column Classes

  • .rw-col-1 through .rw-col-12 - Span 1-12 columns on desktop
  • On mobile (< 768px), all columns automatically span full width (12 columns)
  • Grid gap is set to var(--space-4) (1rem / 16px)

Spacing Tokens

  • var(--space-1) - 0.25rem (4px)
  • var(--space-2) - 0.5rem (8px)
  • var(--space-3) - 0.75rem (12px)
  • var(--space-4) - 1rem (16px)
  • var(--space-6) - 1.5rem (24px)
  • var(--space-8) - 2rem (32px)
  • var(--space-12) - 3rem (48px)
